Slaughter
====

This is an FPS.

## Description
 Slaughter is FPS that you kill all enemies which attack you in a battle filed.

You are impounded and unable to escape. 

Kill them all and survive!

## Usage
W -> forward

A -> left

S -> backward

D -> right

Ctril + W -> dash forwad

Space -> jump

Left Click -> fire

Right Click -> aim
